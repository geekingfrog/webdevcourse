CSS

What is it?

Cascading Style Sheet

  • Another language
  • Makes the html looks good

HTML for CSS

  • Include a doctype: <!DOCTYPE html> to make sure it renders consistently on all browsers.
  • Two very useful tags: <div> and <span>
  • class and id are super useful too.

Syntax

  • Selector: The target for the style.
  • Declaration: a styling element.
  • Don't forget to end each declaration with ;

Using css

Inline declaration

<head>
  <style>
    h1 {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>

Some selectors

tag selector

  • h1 {...} targets all the h1 tags in the document
  • h1 ul li {...} targets all the li tags which have an ul tag as ancestor with a h1 tag as ancestor.

h1 ul li

<h1>
  <p> Very interesting text here
    <ul>
      <li>one</li>
      <li>two</li>
    </ul>
  </p>
</h1>

class and id

  • h1.title for <h1 class="title">
  • h1#title for <h1 id="title">
  • Remember: id should be unique.
  • Prefer use of class
  • Multiple classes & id are ok too
    <div class="button round shiny" id="purchaseButton">

Universal selector

  • * targets every tags.

Group selectors

h1, h2, h3, h4, h5, h6 { color: red;}

All headers are red.

Pseudo class/element

  • a:visited { color: blue; } linked clicked before are blue.
  • a:hover { font-weight: bold; } big links when the mouse is over it.
  • input:focus { border-color: #ccc; } Grey border on form when user is typing stuff.

Attribute selector

  • img[title] targets all images elements with a title.
  • a[href^=http://] targets all links which begins with 'http://'

Which style ? Specificity

  • Tag selector: 1 points
  • Class selector: 10 points
  • ID selector: 100 points
  • Inline selector: 1000 points

examples

  • h1 p.catchphrase 12 points
  • #title img.logo 111 points
  • highest specificity wins
  • Tie ? Last style win.

Recap

  • A css rule has the following form
  • A selector can be any combination of
    • an html tag
    • a class (starts with a .)
    • an id (starts with a #)
    • a pseudo class/element (starts with a :)
  • A rule can apply to multiple selectors: use , between them.

Recap (2)

  • A css rule can have multiple declarations inside the {}.
  • A declaration always end with a ;
  • A declaration has the property on the left, a : and the value after.

Recap (3)

Some basic css properties

  • color: change the color of the text.
  • background-color: change the color of the background.
  • border: shorthand to define borders.
  • font-weight
  • border-radius: to have round corners.
