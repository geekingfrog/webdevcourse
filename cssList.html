Small list of list of css properties

This list contains all properties viewed in the slides and exercises. For everyone of them, one can google 'css {name of the property}' and have the available values and so on.

Box model

  • width and height
  • padding (top/right/bottom/left)
  • border (top/right/bottom/left)
  • margin (top/right/bottom/left)
  • border-radius (top/right/bottom/left)
  • box-sizing

Positionning

  • display
  • visibility
  • position
  • top / right / bottom / left
  • z-index

Colors and text

All colors can be expressed with the following formats (all equivalent):

  • Hex: #4B0082
  • rgb: rgb(75, 0, 130) or rgba(75, 0, 130, 1)
  • hsl: hsl(274.6,100%,25.5%) or hsla
  • browser name: indigo (not always available)

Some properties

  • background-color
  • color
  • border-color
  • font-weight
  • text-align
  • outline

misc

  • pointer
  • list-style-type
  • list-style-style
  • text-decoration
  • content (for generated elements like :after and :before)
