Display

The display property control how an element should be displayed.

block vs inline

A block element will, by default, take 100% width and start on a new line

div1
div2

Examples of block elements

  • <div>
  • headings <h1>
  • <table>
  • <form>
  • <p>
  • ...

An inline element is placed with the other in the flow.

span 1 some random text span 2.

Examples of inline elements

  • <span>
  • <img>
  • <a>
  • <input>

The default behavior can be changed with the display property

display: inline;

Use display: inline-block;

inline-block

  • Cannot set height on inline element
  • Padding top and bottom acts weirdly on inline elements
  • Margin top end bottom have no effects on inline elements

Inline block allow to place an element inline with the same control as a block element.

stop the float

To avoid an element to wrap around a floated element: use clear: both; (can be right, left or both). This prevent any floated element to be placed on the cleared side of the element
