Responsive design

viewport

Add in the html:

<meta name="viewport" content="width=device-width">

This scale the content to the width of the device.

Media queries

  • Styles applied when a certain condition is met.
  • Create breakpoints as a function of the width of the device.

breakpoints

In the css:

These rules only apply when the device's width is below 480px.

mobile first

  • Start your design with a small screen.
  • Increase the screen's size untill it looks really ugly.
  • Create a breakpoint at this size.
  • Rinse and repeat.

Desktop first

The opposite of mobile first: you start from a large screen and reduce the size.
