float and height

  • A floating element does not count in the height of its container.
  • This can be fixed with overflow: hidden on the container
  • Another method is to add an empty div with clear: both; after the last floated element.

box-sizing

Specify how to compute the width of an element.

  • content-box width = content's width (default)
  • border-box width = content's width + padding (left and right) + border (left and right)
