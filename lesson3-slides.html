float and height

  • A floating element does not count in the height of its container.
  • This can be fixed with overflow: hidden on the container
  • Another method is to add an empty div with clear: both; after the last floated element.

To fix it, use box-sizing

box-sizing

Specify how to compute the width of an element.

  • content-box width = content's width (default)
  • border-box width = content's width + padding (left and right) + border (left and right)

Be careful to margins though.

Positionning

The property position control how an element should be placed. It can have the following values

  • static (default)
  • relative
  • absolute
  • fixed
  • inherit

position: relative;

  • Does nothing visible
  • The element can now be nudged with top/left/bottom/right
  • The element now respects the z-index property (altitude)
  • It is now the reference for all children with position: absolute;

position: absolute;

position: fixed;

Same as position: absolute, but the origin is always the top left corner of the window

See examples

hiding elements

Two ways: display: none; or visibility: hidden;

display: none;

  • As if the element is not present at all in the page
  • Doesn't take space
  • Ignored by the browser's rendering engine

visibility: hidden;

  • The element turns invisible
  • But it's still here and take space (margin, border, padding and dimensions)
